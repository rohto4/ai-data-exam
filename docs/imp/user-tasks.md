# User Tasks

このファイルは、ユーザーが実際に確認・操作する作業だけを置く。

**最終更新**: 2026-05-06（プロンプト3完了・動作確認手順追加）

---

## 動作確認手順

### 1. 開発サーバー起動

```bash
npm install
npm run dev
```

- ブラウザで http://localhost:5173 にアクセス
- ダッシュボードが表示されれば成功

### 2. 基本的な動作確認

#### ダッシュボード
- [ ] 全体進捗率が表示される（0%から開始）
- [ ] 10章の章一覧が表示される
- [ ] 各章に6つの節が表示される
- [ ] 節をクリックして詳細画面に遷移

#### 節詳細 → クイズ
- [ ] 学習内容（テキスト）が表示される
- [ ] 「クイズを始める」ボタンをクリック
- [ ] 5問の4択クイズが表示される
- [ ] 回答後、解説が表示される
- [ ] 最後の問題終了後、節詳細に戻る
- [ ] 完了した節にチェックマークが付く

#### 復習モード
- [ ] ヘッダーの「復習」をクリック
- [ ] （誤答があれば）誤答問題がランダムに表示される
- [ ] 「理解した」ボタンで復習完了

#### 設定画面
- [ ] ヘッダーの「設定」をクリック
- [ ] テーマ切り替え（ライト/ダーク/システム）
- [ ] 「エクスポート」でJSONファイル保存
- [ ] 「インポート」でJSONファイル読み込み
- [ ] 「すべて削除」でデータリセット

### 3. データ永続化の確認

```bash
# 1. ブラウザでアプリを開き、いくつかの節を完了させる
# 2. ブラウザを閉じる
# 3. 再度 npm run dev で起動
# 4. ブラウザで http://localhost:5173 にアクセス
# 5. 前回の進捗が保持されていれば成功
```

### 4. テスト実行

```bash
# 全テスト実行（30テスト）
npm test

# テスト結果が全てパスすることを確認
# ✓ src/test/storage.types.test.ts (2 tests)
# ✓ src/test/storage.test.ts (12 tests)
# ✓ src/test/schema.validation.test.ts (16 tests)
```

### 5. ビルド確認

```bash
# 本番ビルド
npm run build

# ビルド成功後、dist/ ディレクトリが生成されることを確認
# dist/
#   ├── index.html
#   └── assets/
#         ├── index-xxx.css
#         └── index-xxx.js
```

### 6. ビルド済みアプリの確認

```bash
# ビルド済みファイルをプレビュー
npm run preview

# ブラウザで http://localhost:4173 にアクセス
# 開発版と同様に動作することを確認
```

## トラブルシューティング

### ポートが使用中の場合
```bash
# 別のポートで起動（例: 3000）
npx vite --port 3000
```

### データをリセットしたい場合
1. ブラウザの開発者ツールを開く（F12）
2. Application/アプリケーションタブ → Local Storage
3. `ai-exam-progress`, `ai-exam-history`, `ai-exam-settings` を削除
4. ページをリロード

または設定画面の「すべて削除」ボタンを使用

---

## 注意

- secret、token、Cookie、未公開の認証情報をドキュメントに残さない。
- 学習教材、問題文、解説文を入れる時は、著作権と出典整理を先に確認する。
