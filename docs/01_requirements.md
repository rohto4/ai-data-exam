# 01 Requirements: 画面遷移図と機能一覧

> 2027年新設「プロフェッショナルデジタルスキル(データ・AI)試験」学習サイトの要件定義

---

## 概要

本アプリは、2027年新設の「プロフェッショナルデジタルスキル(データ・AI)試験」対策学習サイトです。
章 > 節 > クイズ の階層構造で学習を進め、LocalStorageで学習進捗と誤答履歴を永続化します。

## 画面詳細

### 1. ダッシュボード（/）

**機能**:
- 全体の学習進捗率表示（円グラフ or バー）
- 最後に学習した章・節の表示 + 「続きから学習」ボタン
- 全章一覧へのリンク
- 復習モードへのリンク（誤答があれば残件数をバッジ表示）
- 設定画面へのリンク

**表示要素**:
- アプリタイトル: 「データ・AI試験対策」
- 全体進捗: `学習済み: X / Y 節 (Z%)`
- クイズ回答状況: `正答率: A%`
- 復習ボタン: 「誤答 N件を復習」
- 章リスト: 章名 + 節数 + 進捗率

### 2. 章一覧画面（/chapters）

**機能**:
- 全10章のカード表示
- 各章の進捗率表示
- クリックで章詳細画面へ遷移

**表示要素**:
- 章番号 + タイトル
- 進捗バー（節の読了率）
- ステータスラベル（未開始 / 学習中 / 完了）

### 3. 章詳細画面（/chapter/:id）

**機能**:
- 章タイトル + 概要表示
- 所属する節のリスト表示
- 各節の読了フラグ表示
- 節クリックで節詳細画面へ

**表示要素**:
- 章タイトル
- 章の進捗率（%）
- 節リスト: 節番号 + タイトル + 読了アイコン + クイズ数

### 4. 節詳細画面（/section/:id）

**機能**:
- 学習コンテンツ（テキストベース、階層構造）
- 「読了マーク」ボタン
- 「クイズを受ける」ボタン
- 前後の節へのナビゲーション

**表示要素**:
- パンくずリスト: 章 > 節
- 節タイトル
- 学習コンテンツ（Markdown風レイアウト）
- アクションバー: [読了マーク] [クイズ開始]
- ナビゲーション: [前の節] [次の節]

### 5. クイズ画面（/quiz/:sectionId）

**機能**:
- 4択問題を1問ずつ表示
- 回答選択後、即座に正誤判定
- 解説の表示/非表示切り替え
- 次の問題へ進む
- 最後の問題終了後、結果画面（正答数/誤答数）
- 誤答した問題はLocalStorageに記録
- 1周終了後、再チャレンジ or 節詳細へ戻る

**表示要素**:
- 問題番号（N問目 / M問）
- 問題文
- 選択肢4つ（A, B, C, D）
- 回答後: 正解表示（緑/赤）+ 解説
- 次へボタン
- 結果画面: 正答数、誤答数、正答率、各問の正誤リスト

### 6. 復習モード画面（/review）

**機能**:
- 誤答履歴から問題をランダム抽出
- 最大50問までのフィルタ
- 章/節による絞り込み
- クイズ画面と同じUIで出題
- 正解したら誤答履歴から削除

**表示要素**:
- 絞り込みフィルタ: 全章 or 章選択
- 問題数表示: `N問の誤答があります`
- クイズUI（クイズ画面と共通）
- 結果画面: 今回の正答数 + 残り誤答数

### 7. 設定画面（/settings）

**機能**:
- 学習データのJSONエクスポート
- 学習データのJSONインポート
- 全データのリセット
- LocalStorageの容量確認

**表示要素**:
- エクスポートボタン → JSONファイルダウンロード
- インポート: ファイル選択 → バリデーション → 上書き確認
- リセットボタン → 確認ダイアログ → 全削除
- ストレージ使用量

## ユーザーストーリー

```
As a 試験受験予定者,
I want 章ごとに学習内容とクイズを一貫して進められる,
So that 体系的に知識を習得し、弱点を確認できる.

As a 復習者,
I want 間違えた問題だけを繰り返し解ける,
So that 苦手分野を重点的に克服できる.

As a 学習者,
I want 学習進捗を可視化し、データをバックアップできる,
So that モチベーション維持と環境移行時の継続が可能になる.
```
