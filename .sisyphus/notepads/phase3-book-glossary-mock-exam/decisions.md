# Decisions

## 2026-05-07
- 正本候補は `data/syllabus.json` に統一し、`data/syllabus_v2_structure.json` は参照用に隔離する。
- 教本・用語・模試は順次実装し、まずはテスト整合を回復してから次工程へ進む。
- `src/test/schema.validation.test.ts` は現行データの実態に合わせ、`300問=60節×5問` という含意を削除し、`quizzes.json` の sectionId 単位で 5 問であることのみを検証する方針にした。
- Chapter拡張は一括ではなく分割実施とし、今回は `data/content/ch01/*.md` のみを更新対象に固定した。
- Chapter 1 の各節は構成テンプレートを統一し、`[[termId]]` を埋め込んだ学習本文へ再構成した上で、長さ・見出し・マーカー数をスクリプトで検証する方針を採用した。
- Phase3 task 3 は単節単位で進め、今回は `data/content/ch05/sec04.md` のみを更新し、他章・他節には触れない分割実施方針を維持した。
- `ch05/sec04.md` は既存見出し構成（線形回帰/正則化回帰/非線形回帰/アンサンブル学習/バイアスとバリアンス）を保持したまま本文密度を増補する方針にした。
- 次の小分け実装として `data/content/ch05/sec05.md` のみを更新対象に固定し、他章・他節は変更しない方針を継続した。
- `ch05/sec05.md` は既存構成（k-means/階層的クラスタリング/DBSCAN/次元削減）を保持しつつ、距離指標・連結基準・手法比較の説明を増補する方針を採用した。
